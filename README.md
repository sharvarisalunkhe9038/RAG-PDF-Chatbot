# 🤖 RAG PDF Chatbot

An intelligent chatbot that allows users to upload PDF/DOCX files and ask questions based on document content using Retrieval-Augmented Generation (RAG).

## 🚀 Features
- Upload PDF/DOCX files
- Semantic search with FAISS
- Context-based answers using LLaMA 3 (Groq)
- Chat-style UI

## 🧠 Tech Stack
- Python
- LangChain
- FAISS
- Sentence Transformers
- Streamlit
- Groq API

## ▶️ Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
